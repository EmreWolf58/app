# Order Management System (Learning Project)

This repository is a learning-focused full-stack .NET project.

## Tech Stack
- ASP.NET Core Web API
- ASP.NET Core MVC
- SQL Server + Entity Framework Core
- Docker & Docker Compose
- Redis, Hangfire, RabbitMQ (later)

## Goals
- Build a clean solution structure (API + MVC + Core + Infrastructure)
- Practice RESTful API design
- Deploy to IIS and Docker
- Add caching, background jobs, and messaging
- Set up CI pipeline (GitHub Actions)
